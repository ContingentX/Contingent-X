B) Installation

Installing the countdown timer requires the following steps.

  1. Including the css file.
  2. Including the javascript files.
  3. Specifying the html tags of the timer within your page.
  4. Instantiating a javascript function call with any optional parameters.

Below describes these steps in more detail.

In step 1, to include css files, simply specify the following lines of code within your <head> tag:

<link href='http://fonts.googleapis.com/css?family=Open+Sans:400,800,300,600' rel='stylesheet' type='text/css'>
<link rel="stylesheet" type="text/css" href="circular-countdown.css"/>

In step 2, you will need to include the jquery's library files, and the timer's file jquery.circular-countdown.js which is located in the js directory. The code below shows the include in a html page: 

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>
<script type="text/javascript" src="js/jquery.circular-countdown.js"></script>

Note: The css, js files are the compressed production version of the file. If you intend to modify the countdown timer's source code, you will need to look at files in source folder and it is recommended that you minify the modified file afterwards.

In step 3, to specify the tags of the countdown timer itself, refer to the example in the index.html file. In that file, you'll see a block of tag starting and ending with: 

<div class="countdown"></div>

You can use this as a template and copy and paste it onto your own page.

Finally in step 4, you will need to instantiate a javascript function call to get the countdown timer initialized. The example code looks like this below: 

<script type="text/javascript">
  $(document).ready(function() {
    $(".countdown").countdownTimer (
      startDate:"2014/08/01 12:30:00",  //Year/Month/Day Hours:Minutes:Seconds
      endDate:"2017/12/25 12:30:00",
      timeZone:0,
      past:false,
      
      //Show-hide day, hour, minute, second
      showDay:true,
      showHour:true,
      showMinute:true,
      showSecond:true,
      
      //Margin between circles
      margin:10,
      
      //Diameters
      dayDiameter:160,
      hourDiameter:160,
      minuteDiameter:160,
      secondDiameter:160,
      
      //Circle BG width
      dayBgWidth:23,
      hourBgWidth:23,
      minuteBgWidth:23,
      secondBgWidth:23,
      
      //Circle width
      dayCircleWidth:23,
      hourCircleWidth:23,
      minuteCircleWidth:23,
      secondCircleWidth:23,
      
      //Circle BG color
      dayBgColor:"#ccc",
      hourBgColor:"#ccc",
      minuteBgColor:"#ccc",
      secondBgColor:"#ccc",
      
      //Circle color
      dayCircleColor:"#e8065c",
      hourCircleColor:"#e8065c",
      minuteCircleColor:"#e8065c",
      secondCircleColor:"#e8065c",
      
      //Counter font size
      dayCounterFontSize:32,
      hourCounterFontSize:32,
      minuteCounterFontSize:32,
      secondCounterFontSize:32,
      
      //Text font size
      dayTextFontSize:11,
      hourTextFontSize:11,
      minuteTextFontSize:11,
      secondTextFontSize:11,
      
      //Counter font color
      dayCounterFontColor:"#333",
      hourCounterFontColor:"#333",
      minuteCounterFontColor:"#333",
      secondCounterFontColor:"#333",
      
      //Text font color
      dayTextFontColor:"#666",
      hourTextFontColor:"#666",
      minuteTextFontColor:"#666",
      secondTextFontColor:"#666",
      
      //Texts
      dayText:"DAYS",
      hourText:"HOURS",
      minuteText:"MINUTES",
      secondText:"SECONDS",
      
      //Texts top margin
      dayTextMarginTop:0,
      hourTextMarginTop:0,
      minuteTextMarginTop:0,
      secondTextMarginTop:0,
      
      //Timer on finish function
      onFinish:function(){}
    );
  });
</script>

This will instantiate the timer with the selected time.

Also, there is another optional attribute named past. If you want to countup the timer from a past time, add it to javascript function like that: , past: true

C) Plugin Parameters

Attribute Properties Description
startDate string Specifies the start date to countdown.
Date Format: Year/Month/Day Hours:Minutes:Seconds
endDate string Specifies the end date to countdown.
Date Format: Year/Month/Day Hours:Minutes:Seconds
timeZone number - Default: 0 Specifies the time zone. GMT number.
http://www.timeanddate.com/time/map/
past boolean - Default: false Specifies if the timer count-downs or count-ups. Set to true or false.
showDay boolean - Default: true Specifies if the days are visible. Set to true or false.
showHour boolean - Default: true Specifies if the hours are visible. Set to true or false.
showMinute boolean - Default: true Specifies if the minutes are visible. Set to true or false.
showSecond boolean - Default: true Specifies if the seconds are visible. Set to true or false.
